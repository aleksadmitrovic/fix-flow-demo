# FixFlow

A workspace and ticket management application for managing support requests between clients and technicians.

## Live Demo

[https://fix-flow-demo.vercel.app](https://fix-flow-demo.vercel.app)

## Features

- Role-based access control (Admin, Technician, Client)
- Ticket lifecycle management — create, assign, close, and reopen tickets
- Workspace management with invite codes
- Pagination for ticket listings
- Authentication with NextAuth

## Tech Stack

- **Next.js 14** — App Router, Server Actions
- **TypeScript**
- **Prisma** — ORM with PostgreSQL
- **HeroUI** — component library
- **Zod** — schema validation
- **NextAuth** — authentication

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
